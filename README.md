# Geospatial Workflows with Open Source Tools

A practical guide to common GIS workflows using Python, GDAL/OGR, and other open source tools.

## Purpose
This repository serves as a reference guide for geospatial professionals looking to accomplish common GIS tasks using open source tools as alternatives to proprietary software like FME.

## Tech Stack
- Python 3.9+
- GDAL/OGR
- GeoPandas
- Rasterio
- Shapely
- SQLite/PostGIS (optional)

## Workflows Covered
1. Format Conversion
2. Cooredinate Transformation
3. Spatial Filtering
4. Attribute Joins
5. Batch Processing
6. Geometry Validation
